About the OPeNDAP ncml_module package version 0.9.0

This is the OPeNDAP NcML Data Handler. It is used along with the OPeNDAP Hyrax server

To install ncml_handler please read the Readme file provided with this distribution. After installation, you will need to run "bes-ncml-data.sh" to tell Hyrax to load this module. You will also need to restart Hyrax (both OLFS and BES). See OPeNDAP web site for more information.
